# PostRenderCleaner

Cleans up artifacts and optimizes files after rendering.

## Overview

PostRenderCleaner processes rendered media files to remove artifacts, optimize file size, and prepare them for distribution. It handles tasks such as removing temporary files, optimizing compression, generating derivatives in different formats, and organizing final output files.

## Features

- Artifact detection and removal
- File optimization for different delivery platforms
- Automatic derivative generation
- Project cleanup and organization

## Implementation

- Primary language: Python
- Key technologies: Google Cloud Storage, Google Cloud Functions
- Trigger mechanism: Webhook post-render
